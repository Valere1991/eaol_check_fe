# EAOL Check FE

Angular dashboard for the EAOL IT-domain pilot.

## Local start

```bash
npm install
npm start
```

Open:

```text
http://localhost:4200
```

Backend expected at:

```text
http://localhost:8000
```

Default tenant:

```text
firma-it
```

## Features

- Health check
- Exact license status check
- German IT root-cause analysis demo
- Audit view
- Expired/missing license errors displayed from backend
